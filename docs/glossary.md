# Glossary

## Request

The starting record for an IT need. A request can have related tasks.

## Task

A piece of work assigned to an employee. Tasks can track status, progress, due date, blocked reason, and completion.

## Task Update

An immutable record of task status/progress changes.

## Comment

A note attached to a request for MVP.

## Audit Log

A protected record of an important action, including actor, action, entity type, old value, and new value.

## Role

A named access level assigned to users.

## Permission

A specific action allowed for a role, grouped by module.

## No-Update Item

A task that has not had a progress update for a defined number of days.

## Cancelled Terms For Now

Purchases, MR, products, assets, suppliers, attachments, notifications, and settings are not part of the current system scope.
